Allocate memory for a contact, set the fields of the contacts to the passed arguments and add the contact to the contacts graph.

Function Name:
add_contact_to_graph
Date Written:
13/01/20
Returns
int
Return values
2Success case: Revised xmit rate (and MTVs)
1Success case: contact added to the contacts graph
0Arguments error case
-1Overlapped contact (don't change it)
-2MWITHDRAW error
Parameters
[in]fromNodeThe contact's sender node
[in]toNodeThe contact's receiver node
[in]fromTimeThe contact's start time
[in]toTimeThe contact's end time
[in]xmitRateIn bytes per second
[in]confidenceThe confidence that the contact will effectively materialize
[in]copyMTVSet to 1 if you want to copy the MTV from mtv input parameter. Set to 0 otherwise
[in]mtvThe contact's MTV: this must be an array of 3 elements.
Notes:
  1. This function will change timeContactToRemove if the contact->toTime of the new contact is less than the current timeContactToRemove, in that case timeContactToRemove will be equals to contact->toTime
  2. Set copyMTV to 1 if you want to be compatible with the data gets for example from an interface.
