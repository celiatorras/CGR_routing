Detailed Description

This file provides the implementation of the functions that make this CGR's implementation compatible with ION.

In particular the functions that import the contact plan and the BpPlans. We provide in output the best routes list.

Returns
int
Return values
>= 0Number of contacts converted
-1CGR's contact NULL
-2Memory allocation error
-3Contact not found in the ION's contacts graph
Parameters
